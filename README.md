# Hash Otomasyon

Bu proje, metinlerin çeşitli hash algoritmalarıyla (MD5, SHA1, SHA256, SHA512) kolayca dönüştürülmesini sağlayan, görsel arayüzlü bir Python/Tkinter uygulamasıdır.

## Özellikler
- Modern ve koyu temalı arayüz
- Görsel başlık desteği (banner.png)
- MD5, SHA1, SHA256, SHA512 hash algoritmaları
- Sonuçları kolayca kopyalama

## Kurulum

1. **Gereksinimler:**
   - Python 3.10+
   - pip
   - Pillow (PIL)
   - tkinter (çoğu Python dağıtımında yüklü gelir)

2. **Depoyu klonlayın veya dosyaları indirin:**
   ```bash
   git clone https://github.com/mkati42/hashotomasyon.git
   cd hashotomasyon
   ```

3. **Gereksinimleri yükleyin:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Uygulamayı başlatın:**
   ```bash
   python3 hash.py
   ```

## Notlar
- `banner.png` dosyasını proje klasörüne eklemeyi unutmayın.
- Arayüzde görsel görünmüyorsa dosya adını ve konumunu kontrol edin.
